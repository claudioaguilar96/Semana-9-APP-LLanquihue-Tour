# Semana-9-APP-LLanquihue-Tour
Aplicación de consola en Java para la gestión de un operador de turismo: administra colaboradores (guías, operadores de transporte y proveedores de alojamiento), clientes, tours y reservas, con persistencia en archivos de texto plano , mediante elementos como herencia, relaciones de compocicion , poliformismo, control de entidades e interfaces .
